# WAC<sup>3</sup>

## Development

Start our server using:

```shell
$ npm install
$ npm run server
```

Now you should now be able to visit `http://localhost:8090/`.

### Production use

This app should be easily deployed in [Heroku](https://www.heroku.com/).
